# AI Deep Research Agent

An AI-powered research assistant that performs web-based research on a given topic and generates a structured research report using OpenAI Agents SDK and Firecrawl.

## Overview

This project automates the research process by collecting information from multiple web sources, analyzing the gathered content, and generating a detailed report with key findings and insights.

## Features

* Topic-based web research
* Automated content collection using Firecrawl
* AI-powered research analysis using OpenAI Agents SDK
* Structured research report generation
* Simple Streamlit user interface
* Download research reports in Markdown format

## Tech Stack

* Python
* OpenAI Agents SDK
* Firecrawl
* Streamlit

## Project Workflow

1. User enters a research topic.
2. Firecrawl searches and extracts relevant web content.
3. Research findings are processed and summarized.
4. OpenAI Agent analyzes and enhances the research.
5. Final report is displayed and available for download.

## Installation

```bash
git clone <your-repository-url>
cd <project-folder>
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run deep_research_openai.py
```

## Configuration

Provide the following API keys before running the application:

* OpenAI API Key
* Firecrawl API Key
